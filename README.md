# Site de Tutoriels en Ligne

Un site web simple pour créer et partager des tutoriels en ligne.

## Fonctionnalités

- Créer votre propre chaîne de tutoriels
- Ajouter des tutoriels à votre chaîne
- Voir tous les tutoriels des autres utilisateurs
- Rechercher des tutoriels depuis la page d'accueil

## Technologies utilisées

- HTML
- CSS
- JavaScript (localStorage pour la persistance des données)

## Déploiement

Ce site est déployé sur GitHub Pages.
